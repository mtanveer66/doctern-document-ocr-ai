# Doctern — Frequently Asked Questions

## General

**What is Doctern?**
Doctern is an AI-powered document OCR and table extraction tool. It converts PDFs, scans, and images into clean text, structured tables, and named fields.

**What problem does it solve?**
It eliminates manual data entry from documents — retyping invoices, forms, statements, and reports.

## Capabilities

**What file types are supported?**
PDFs, scanned documents, and common image formats such as PNG and JPG.

**Does it preserve table structure?**
Yes. Doctern keeps rows and columns intact, so values never end up in the wrong cell.

**Can it handle tables without visible borders?**
Yes. It reconstructs borderless tables by analyzing the spatial layout of the content.

**How does it perform on poor-quality scans?**
Doctern uses deep-learning OCR that handles noisy, skewed, and low-resolution documents well.

**Can I extract only specific fields instead of the whole document?**
Yes. Field matching lets you target named values like totals, dates, or reference numbers.

## Output

**What format is the output?**
Structured data — spreadsheet-ready tables and structured JSON.

**Can I use the output in Excel or Google Sheets?**
Yes. Extracted tables are designed to paste directly into spreadsheet software.
